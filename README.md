# Cojeet Android Application

#### This application is a one stop platform for all the COVID-19 or related resources including :
> - COVID Self Checker
> - COVID Statistics
> - Vaccination Statistics
> - Easy communication to nearest Hospital
> - COVID News
> - Edit User Profile
